## **Installation & Setup**

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional for development)

### **Local Setup**
1. **Clone the repository**
   ```bash
   git clone https://github.com/Vasantjv-2005/portfolio-website.git
   cd portfolio-website
   ```

2. **Open in Browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server for better development
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **Live Server (VS Code Extension)**
   - Install Live Server extension in VS Code
   - Right-click on `index.html` and select "Open with Live Server"

## **Configuration**

### **Personalization**
To customize this portfolio for your use:

1. **Update Personal Information**
   - Change name in `index.html` (hero section)
   - Update social media links in contact section
   - Modify email address in contact form

2. **Update Projects**
   - Replace project details in projects section
   - Update GitHub and demo URLs
   - Modify project descriptions

3. **Customize Skills**
   - Update skill percentages in progress bars
   - Add or remove skill cards
   - Modify skill icons

### **Color Scheme**
The color scheme can be easily customized in `style.css`:
- Primary colors: `#ff6b6b` (coral red), `#4ecdc4` (turquoise)
- Background gradients: Dark theme with purple/blue tones
- Accent colors: `#45b7d1` (blue) for additional elements
